Creating an Application ​

When creating an application with Ptah.js, you can follow a series of simple steps. Here is a detailed guide to help you create and launch a Ptah.js application.

Application Instance ​

Create a new application instance using the createPtah function:

js
import { createPtah } from "@ptahjs/core";

const ptah = createPtah({
  id: "ve1",
});

Using Vue Instance ​

Next, associate the Vue instance with the Ptah application using the $vue method:

js
const vueInstance = createApp(App);
ptah.$vue(vueInstance);

Using Modes ​

Ptah.js provides various modes that can be used with the $mode method:

js
import ModeWeb from "@ptahjs/mode-web";
import ModeMobile from "@ptahjs/mode-mobile";
import ModeCanvas from "@ptahjs/mode-canvas";
import Mode3D from "@ptahjs/mode-3d";
import ModeFlow from "@ptahjs/mode-flow";

ptah.$mode(ModeWeb);

Loading Resources ​

You can load various resource types using the load method:

js
ptah.load(ResourceType.SceneProps, []);
ptah.load(ResourceType.Modules, []);
ptah.load(ResourceType.ModuleProps, []);
ptah.load("Custom", [1, 2, 3]);

Mounting the Application ​

Finally, launch the application using the mount method:

js
ptah.mount();

Complete Code Example ​

Here is a complete code example showing how to create and launch a Ptah.js application:

js
import { ResourceType, createPtah } from "@ptahjs/core";
import ModeWeb from "@ptahjs/mode-web";
import { modules, sceneProps } from "./resources";

// Create Vue instance
const vueInstance = createApp(App);

// Create and configure Ptah instance
const ptah = createPtah({
  id: "ve1",
})
  .$vue(vueInstance) // Associate Vue instance
  .$proxyData((data) => reactive(data)) // Set proxy data
  .$mode(ModeWeb) // Use Web mode
  .$load(ResourceType.Modules, modules) // Load module resources
  .mount(); // Launch application

By following these steps, you can easily create and launch a Ptah.js application, and configure and extend it as needed.
