Introduction ​

What is PtahJs? ​

PtahJs is an extensible headless low-code framework. Its core engine is refined from enterprise-level low-code platforms, aiming to simplify the development process and enhance extensibility.

Features: ​

  • Supports Vue3: Seamlessly integrates with the popular Vue3 framework.
  • Designed for Extensibility: The core engine is meticulously designed for easy extension and customization.
  • Component-Based Development: Build the low-code platform by writing components, reducing the learning curve.
  • Built-in Multi-Mode Support: Supports nested components, canvas, 3D views, and flowcharts to meet diverse needs.

Component Nesting ​

PtahJs allows developers to construct applications in a component-based manner, where components can nest within each other to form complex layouts. This approach not only enhances code reusability but also simplifies maintenance.

Canvas Mode ​

With the built-in canvas mode, developers can intuitively drag and drop components to quickly build UI interfaces. Canvas mode supports various layouts and custom styles, making interface design more flexible.

3D View ​

PtahJs includes a 3D view mode, allowing the construction and display of 3D models on the platform. This feature is particularly suitable for applications requiring three-dimensional presentation, such as virtual reality and product showcases.

Flowchart ​

The platform also supports flowchart mode, ideal for constructing applications related to process management and task scheduling. Through flowcharts, developers can visually display and manage various business processes, improving efficiency.

Through these features, PtahJs provides a powerful and extensible low-code development environment, enabling developers to efficiently build a wide range of applications.
