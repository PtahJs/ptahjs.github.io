Built-in Components ​

INFO

Built-in components can be used directly in templates without needing to register them.

<Provider> ​

The <Provider> component provides Ptah functionality to your application. It must be injected with a Ptah instance through the app prop, typically used in the App.vue file.

  • Example
    vue
    <template>
  <Provider :app="app">
    <!-- Your Ptah Application -->
  </Provider>
</template>

<Preview> ​

The <Preview> component is used for content rendering previews. It must be injected with a Ptah instance through the app prop.

  • Example
    vue
    <template>
  <Preview :app="app"></Preview>
</template>

<Workspace> ​

The <Workspace> component is the workspace for content dragging and dropping. It must be used inside the <Provider> component.

  • Example
    vue
    <template>
  <div class="lowcode-workspace">
    <Workspace></Workspace>
  </div>
</template>

<ModulesRenderer> ​

The <ModulesRenderer> component is used for rendering modules. It must be used inside the <Provider> component.

  • Example
    vue
    <template>
  <ModulesRenderer :item="item.items[0]" />
</template>

<SourceModulesWidget> ​

The <SourceModulesWidget> component is used to display a list of modules. It must be used inside the <Provider> component.

  • Example

    vue
    <template>
  <!-- Simplified version -->
  <SourceModulesWidget></SourceModulesWidget>

  <!-- Customized version -->
  <SourceModulesWidget>
    <template #header="item">{{ item.title }}</template>
    <template #sectionHeader="item">{{ item.title }}</template>
    <template #content="item">{{ item.title }}</template>
  </SourceModulesWidget>
</template>

<ScenesWidget> ​

The <ScenesWidget> component is used to display a list of scenes.

  • Example

    vue
    <template>
  <!-- Simplified version -->
  <ScenesWidget></ScenesWidget>

  <!-- Customized version -->
  <ScenesWidget>
    <template #content="item">{{ item.props.title || item.title }}</template>
    <template #empty>No data</template>
  </ScenesWidget>
</template>

<ModulePropsWidget> ​

The <ModulePropsWidget> component is used to display module parameters.

  • Example

    vue
    <template>
  <!-- Simplified version -->
  <ModulePropsWidget></ModulePropsWidget>

  <!-- Customized version -->
  <ModulePropsWidget>
    <template #header="item">{{ item.label }}</template>
    <template #groupHeader="item">{{ item.label }}</template>
    <template #before="item"><span>1</span></template>
    <template #after="item"><span>2</span></template>
    <template #empty>No data</template>
  </ModulePropsWidget>
</template>

<EventsWidget> ​

The <EventsWidget> component is used to display a list of events.

  • Example

    vue
    <template>
  <!-- Simplified version -->
  <EventsWidget type="module"></EventsWidget>

  <!-- Customized version -->
  <EventsWidget>
    <template #content="item">
      <div>
        <button>Configure</button>
      </div>
    </template>
  </EventsWidget>
</template>

<ScenePropsWidget> ​

The <ScenePropsWidget> component is used to display scene properties.

  • Example

    vue
    <template>
  <!-- Simplified version -->
  <ScenePropsWidget></ScenePropsWidget>

  <!-- Customized version -->
  <ScenePropsWidget>
    <template #header="item">{{ item.label }}</template>
    <template #groupHeader="item">{{ item.label }}</template>
    <template #before="item"><span>1</span></template>
    <template #after="item"><span>2</span></template>
    <template #empty>No data</template>
  </ScenePropsWidget>
</template>
